How to set program to run on start

Windows

1. Press (Windows Key + R) to open the Run dialog box and type in "shell:startup" and press enter. 2. A folder will open, right click and select "New" and then "Shortcut". 3. Right click the new shortcut and click "Properties". 4. In the "Target" field, type in the path to SafeGuard.app (DEFAULT: C:\Users\YOUR-USER\Python-SafeGuard\SafeGuard.app). 5. Click "Apply" and then "OK". 6. You're done! SafeGuard will now run on startup.
